# 冰髓争夺战 · 积分计算

> 梳理对象：冰髓争夺战**战斗阶段（EICSFight）**玩家积分（`PlayerBattleScore`，每轮重置）的计算公式与落点。
> 统一入口 `AddPlayerIceContestBattleScore`（`game/wmap/internal/ctl_union_center.go:567`）→ 仅在 `EICSFight` 阶段调 `icm.AddPlayerBattleScore`；驻防类积分走 `settlePlayerBuildScore`（`ctl_union_center.go:628`）直接累加。

---

## 符号说明

| 符号 | 含义 |
|------|------|
| `d` | 被本部队击杀的目标士兵数（阵亡数） |
| `P0 / P1` | 目标方初始战力 / 剩余战力 |
| `ΔP` | 损失战力 = `P0 - P1` |
| `r` | 阵亡修正系数，查 `troopslossmap` 表（入参：己方兵种 / 目标兵种 / `ΔP÷P0`），`GetDeadRate`（`gdconf/def.go:1087`） |
| `k` | 击杀积分系数 = D2Const `IceMapGame_kill_soldier_point_parameter` |
| `T_xxxx` | 目标积分 = `IceMarrowWartarget` 表 `TargetPoints`（按 playbook + 目标编号取） |
| `pi / PΣ` | 集结中成员 i 的战力 / 集结总战力 |
| `m` | 驻防分钟数 = `⌊驻防秒数 ÷ 60⌋`，且 `m ≥ 1` |

目标积分编号（`expedition/ice_contest_mgr.go:13`）：

| 编号 | 含义 |
|------|------|
| 1001 | 击杀士兵（概念项，实际用公式不读 TargetPoints） |
| 1002 | 阵亡士兵 |
| 1003 | 攻击建筑 |
| 1004 | 防守建筑 |
| 1005 | 防御水晶塔 |

## 公式总表

| 积分项 | 适用对象 | 计算公式 | 说明 |
|--------|----------|----------|------|
| 击杀基础分 `S_base` | 公共 | `d × √(ΔP × r ÷ d)`（d>0），否则 `0` | 击杀类积分的基础值 |
| 击杀士兵分 | 玩家部队 | `S_base × k` | 乘击杀系数 k |
| 击杀士兵分 | 集结部队（每成员 i） | `(pi ÷ PΣ) × S_base × k` | 按战力占比分摊；⚠️ 现代码漏乘 k |
| 阵亡士兵分 | 玩家 / 集结成员各自 | `d_self × T_1002` | 按自己实际阵亡数，不分摊 |
| 攻击建筑分 | 玩家 | `T_1003` | 每人每轮只加一次 |
| 驻防·守建筑分 | 玩家 | `T_1004 × m` | 不足 1 分钟不计 |
| 驻防·水晶塔分 | 玩家 | `T_1005 × m` | 同上 |
| **单轮总积分** | 玩家 | `ΣS_kill + ΣS_dead + S_atk + ΣS_def` | 各项累加 |

## 公式细节

### 0. 击杀基础分 `S_base`

`clacForgottenPvpKillScore`（`ctl_battle_settle.go:224`）：

```
              ┌  d × √( ΔP × r ÷ d )      ,  d > 0
  S_base  =   │
              └  0                        ,  d = 0
```

> 集结目标会遍历每支成员部队累加 `d`、`P0`、`P1`。
> 函数内 `killScoreFactor = Expedition_Building_KillPoints` 仅打日志，不参与计算。

### 1. 击杀士兵分

- 玩家部队（`HandleIceContestPTroopBattleScores:349` / `SettlePTroopAtkUnionCenterBuilds:424`）：
  ```
  S_kill = S_base × k
  ```
- 集结部队按战力占比分摊给每个成员 i（`HandleIceContestRTroopKillSodierScores:455`）：
  ```
  S_kill_i = (pi ÷ PΣ) × S_base × k      （所有成员之和 = S_base × k）
  ```
  > ⚠️ **现代码漏乘 k**：实际为 `(pi÷PΣ) × S_base`。第 459 行 `total := baseScore × int64(scoreCoef)` 既未在循环中使用，且 `int64(scoreCoef)` 对小于 1 的系数会被截断为 0，是死代码。上式为「应有」公式。

### 2. 阵亡士兵分

玩家 / 集结成员各按自己实际阵亡数算，**不分摊**（玩家部队 `:361`；集结 `HandleIceContestRTroopDeadSodierScores:477`）：

```
S_dead = d_self × T_1002
```

### 3. 攻击建筑分

每名玩家每轮只加一次（玩家部队 `IsPlayerGetAttackScore` 判重 `:408`；集结遍历成员逐个判重 `:534`）：

```
S_atk = T_1003
```

### 4. 驻防分（守建筑 / 水晶塔）

`settlePlayerBuildScore`（`:628`）按驻防时长（分钟）计：

```
S_def = T_id × m      id = 1004 守建筑 / 1005 水晶塔
m = ⌊驻防秒数 ÷ 60⌋ ，m ≥ 1（不足 1 分钟不计）
```
