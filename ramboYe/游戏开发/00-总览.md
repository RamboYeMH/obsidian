# 🎮 游戏开发管理总览

> SLG 服务端开发工作的中枢。下面的列表用 Dataview 插件自动汇总（需在「设置 → 第三方插件」启用 Dataview）。
> 没装 Dataview 也没关系，手动点下面的文件夹链接即可。

## 📁 快速入口

- [[版本]] — 各版本（v1044 / v1045 …）的内容与进度
- [[需求]] — 需求清单
- [[Bug]] — Bug 单（如 H5-9953）
- [[模块设计]] — 模块/玩法设计笔记
- [[工作日志]] — 每日工作记录

---

## 🚧 进行中的需求

```dataview
TABLE 版本 AS "版本", 状态, 优先级
FROM "游戏开发/需求"
WHERE 状态 != "已完成"
SORT 优先级 DESC
```

## 🐞 未关闭的 Bug

```dataview
TABLE 单号, 版本, 严重度, 状态
FROM "游戏开发/Bug"
WHERE 状态 != "已修复"
SORT 严重度 DESC
```

## 📅 最近的工作日志

```dataview
LIST
FROM "游戏开发/工作日志"
SORT file.name DESC
LIMIT 7
```
