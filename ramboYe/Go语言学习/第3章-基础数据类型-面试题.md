# 第3章-基础数据类型-面试题

关联：[[第3章-基础数据类型]]、[[00-Go语言学习索引]]

## 基础题

### 1. Go 的基础数据类型主要包括哪些？
参考答案：数字类型、字符串类型和布尔类型。本章还讲常量规则。

### 2. `byte` 和 `rune` 分别是什么？
参考答案：`byte` 是 `uint8` 的别名，强调原始字节；`rune` 是 `int32` 的别名，表示 Unicode 码点。

### 3. `int` 和 `int32` 是同一种类型吗？
参考答案：不是。即使某个平台上 `int` 也是 32 位，它和 `int32` 仍是不同类型，需要显式转换。

### 4. Go 整数溢出会 panic 吗？
参考答案：不会。结果按目标类型宽度截断，例如 `uint8(255)+1` 变成 0。

### 5. 为什么普通数量通常使用 `int` 而不是 `uint`？
参考答案：`len` 返回 `int`，使用 `uint` 容易引入类型转换和逆序循环 bug；`uint` 更适合位运算、二进制协议等场景。

### 6. Go 的取模 `%` 对负数有什么规则？
参考答案：结果符号和被取模数一致，例如 `-5 % 3` 是 `-2`。

### 7. 整数除法和浮点除法有什么区别？
参考答案：整数除法向 0 截断，例如 `5/4` 是 1；浮点除法保留小数，例如 `5.0/4.0` 是 1.25。

### 8. `&^` 位运算是什么意思？
参考答案：AND NOT，按位置零。`x &^ y` 会把 y 中为 1 的 bit 在 x 中清零。

### 9. `float32` 和 `float64` 通常优先选哪个？为什么？
参考答案：通常选 `float64`。它精度更高，累计误差更小；`float32` 大约只有 6 位十进制精度。

### 10. NaN 可以用 `==` 判断吗？
参考答案：不能。NaN 和任何值都不相等，包括自己。应使用 `math.IsNaN`。

### 11. Go 中 bool 能和数字互转吗？
参考答案：不能隐式互转。需要显式写 if 或封装函数，比如 true 转 1、false 转 0。

### 12. `&&` 和 `||` 有什么重要特性？
参考答案：短路求值。左侧已能决定结果时，右侧不会执行。

## 字符串题

### 13. Go 字符串是什么？
参考答案：不可变的字节序列。通常保存 UTF-8 文本，但也可以保存任意字节。

### 14. `len(s)` 返回字符数吗？
参考答案：不是，返回字节数。UTF-8 中文字符通常占 3 个字节。

### 15. `s[i]` 返回什么？
参考答案：返回第 i 个字节，不一定是第 i 个字符。

### 16. 字符串能原地修改吗？
参考答案：不能。字符串不可变，`s[0] = 'x'` 不能编译。

### 17. 如何统计字符串中的 Unicode 字符数？
参考答案：可以用 `utf8.RuneCountInString(s)`，或者 `for range s` 手动计数。

### 18. `for i, r := range s` 中 i 和 r 分别是什么？
参考答案：`i` 是当前 rune 的字节偏移，`r` 是解码后的 Unicode 码点。

### 19. `[]byte(s)` 和 `[]rune(s)` 有什么区别？
参考答案：`[]byte(s)` 得到 UTF-8 字节；`[]rune(s)` 得到 Unicode 码点序列。

### 20. `string(65)` 的结果是什么？
参考答案：`"A"`，不是 `"65"`。整数转字符串表示生成对应 Unicode 码点的字符串。

### 21. 数字转字符串应该用什么？
参考答案：用 `strconv.Itoa`、`strconv.FormatInt`，或用 `fmt.Sprintf` 格式化。

### 22. 字符串转整数应该用什么？
参考答案：用 `strconv.Atoi` 或 `strconv.ParseInt/ParseUint`，并处理返回的 error。

### 23. `strings`、`bytes`、`strconv`、`unicode` 各负责什么？
参考答案：`strings` 操作字符串，`bytes` 操作 `[]byte`，`strconv` 做字符串和基础类型转换，`unicode` 按 Unicode 规则处理 rune 分类和大小写。

### 24. 为什么大量字符串拼接建议用 buffer？
参考答案：字符串不可变，反复拼接可能产生多次分配和复制；`bytes.Buffer` 可以动态增长并减少分配。

## 常量题

### 25. 常量什么时候计算？
参考答案：编译期计算。

### 26. 常量可以作为数组长度吗？
参考答案：可以，因为数组长度必须是编译期常量。

### 27. `iota` 是什么？
参考答案：常量生成器，在一个 `const` 声明块中从 0 开始，每一行常量声明递增 1。

### 28. `iota` 常用于什么？
参考答案：枚举值和 bit flag，例如星期、权限标志、网络接口标志。

### 29. 什么是无类型常量？
参考答案：还没有确定具体 Go 类型的常量。它在编译期有高精度，直到赋值或使用时再转换为具体类型。

### 30. 无类型常量的默认类型是什么？
参考答案：整数默认 `int`，字符默认 `rune`，浮点默认 `float64`，复数默认 `complex128`，字符串默认 `string`，布尔默认 `bool`。

## 代码阅读题

### 31. 下面输出什么？
```go
var u uint8 = 255
fmt.Println(u + 1)
```
参考答案：输出 `0`，因为 `uint8` 溢出后高位被截断。

### 32. 下面输出什么？
```go
fmt.Println(5/4, 5.0/4.0)
```
参考答案：输出 `1 1.25`。

### 33. 下面输出什么？
```go
s := "Hello, 世界"
fmt.Println(len(s))
```
参考答案：输出 `13`，因为 `len` 统计 UTF-8 字节数。

### 34. 下面代码有什么风险？
```go
for i := uint(len(xs)-1); i >= 0; i-- {
    fmt.Println(xs[i])
}
```
参考答案：`i >= 0` 对无符号数永远为真，循环不会按预期结束，还可能越界 panic。

### 35. 下面输出什么？
```go
nan := math.NaN()
fmt.Println(nan == nan)
```
参考答案：输出 `false`。

### 36. 下面输出什么？
```go
fmt.Println(string(0x4eac))
```
参考答案：输出 `京`。

### 37. 下面代码为什么安全？
```go
if s != "" && s[0] == 'x' {
}
```
参考答案：`&&` 短路，空字符串时不会执行 `s[0]`。

### 38. 下面代码中 x 的类型是什么？
```go
x := 0.0
```
参考答案：`float64`。

## 实战追问

### 39. 如何正确反转包含中文的字符串？
参考答案：不能按 byte 反转，应先转为 `[]rune`，反转 rune slice，再转回 string。

### 40. 判断两个字符串是否由相同字符组成时要注意什么？
参考答案：要明确按 byte 还是 rune 判断。处理 Unicode 文本时应按 rune 计数，通常用 `map[rune]int`。

### 41. 为什么 JSON、网络协议处理中经常看到 `[]byte`？
参考答案：I/O 和协议传输面向字节流，`[]byte` 可变且避免过多字符串转换。

### 42. 面试中如何解释 Go 字符串和 UTF-8 的关系？
参考答案：字符串只是不可变字节序列；Go 源码和文本惯例使用 UTF-8；`range` 会按 UTF-8 解码为 rune。

### 43. 什么时候会使用复数类型？
参考答案：数学计算、信号处理、分形图像等需要复平面的场景。普通业务开发较少使用。

### 44. 为什么无类型常量有价值？
参考答案：它能在编译期保持高精度，并能自然用于不同目标类型，减少不必要的显式转换。
